Privacy Policy & Terms of Service

By using this Resume Generator service, you acknowledge and accept these terms and our data processing practices.

1. What Data We Process

When you use this Resume Generator, we temporarily process the following personal data:

  • Resume content: Name, contact information (email, phone, location), work experience, education, skills, projects, professional bio
  • Photos: Professional photos you upload (processed as base64-encoded images)
  • Custom preferences: Selected theme, color scheme, section customizations

2. How We Process Your Data

HTML Preview Generation:

  • Your resume data is sent to our server for HTML rendering
  • The server generates a formatted HTML preview using your selected theme and colors
  • The HTML is returned to your browser for display
  • No data is saved to disk or database during this process

PDF Export:

  • When you click "Export PDF", your data is sent to the server
  • A temporary PDF file is created using Puppeteer (headless browser)
  • The PDF is immediately sent to you as a download
  • The temporary file is deleted from the server immediately after download

Privacy Guarantee: We do NOT store your resume data permanently. All processing is temporary and data is deleted immediately after use.

3. Data Retention

  • Resume data: Processed in-memory only, never written to permanent storage
  • Temporary PDF files: Stored for less than 1 second (deleted after download completes)
  • Your browser data: All your resume data remains in your browser's local memory until you close the page

4. Analytics & Tracking

Matomo Analytics:

This website uses Matomo (self-hosted analytics) to understand how visitors use our service. Matomo collects:

  • IP addresses (anonymized)
  • Pages visited and time spent
  • Browser type and device information
  • Referring websites

This data helps us improve the service. You can opt-out of tracking using browser extensions like "Do Not Track" or ad blockers.

5. Legal Basis for Processing

We process your personal data based on:

  • Consent: By using this service, you consent to the temporary processing of your data for resume generation
  • Legitimate Interest: We have a legitimate interest in providing this service and improving it through analytics

6. Your Rights (GDPR)

Under GDPR, you have the right to:

  • Access: Request what data we have (in our case: none stored permanently)
  • Rectification: Correct any inaccurate data (you control all data in your browser)
  • Erasure: Request deletion (simply close the browser tab - no data persists)
  • Data Portability: Export your data (use "Export JSON" button)
  • Withdraw Consent: Stop using the service at any time
  • Object to Processing: Do not use the service if you object to temporary server-side processing

7. Data Security

  • All data transmission should occur over HTTPS (encrypted connections)
  • Server processing happens in isolated sessions with no cross-user data exposure
  • No data is logged to files or databases
  • Temporary files are immediately deleted after use

8. Data Controller

The data controller for this service is the operator of this Resume Generator instance.

For privacy inquiries or to exercise your rights, please contact the service operator through the GitHub repository linked on the main page.

9. International Data Transfers

Your data is processed on the server where this instance is hosted. No data is transferred to third parties or other countries, except:

  • Matomo analytics data (stored on our self-hosted Matomo instance)
  • CDN resources (Font Awesome, Google Fonts) loaded by your browser
